About

this project is a simple recipe website that consists of:
1- A homepage containing links to four recipe pages.
2- Four individual recipe pages each containing an image, a description, a list of ingredients, and step-by-step preparation instructions to the recipe.

What I learned

Building this project helped with:
1- using the command line a lot and getting more comfortable with it.
2- practicing the html we learned so far.
3- getting more used to git and gitHub specially in trying to apply what i learned about the correct way of writing a commit message.
